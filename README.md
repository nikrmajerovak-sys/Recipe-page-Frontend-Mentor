# Frontend Mentor | Recipe Page Challenge


### The Challenge
Build out this recipe page following the design provided by Frontend Mentor. Users should be able to view a recipe with preparation time, ingredients, instructions, and nutrition information. The page should be responsive and visually appealing.

## Technologies Used
- HTML5  
- CSS3 (Flexbox, Variables, Responsive Design) 

## Features
- Fully responsive layout for desktop and mobile
- Styled lists with custom bullets and numbered steps
- Use of CSS variables for easy theming
